# prisma

Six spectral and granular effects, one per page, one knob each. Simple, powerful audio transformation.

## Description

Prisma is a minimal yet expressive effect processor featuring six distinct spectral and granular audio effects, each accessible on its own page. With a single dedicated control knob per effect, Prisma strips away complexity to reveal pure sonic transformation. Features include granular morphing, spectral freezing, reactive graining, modal resonance, and more.

Perfect for sound design, texture creation, and real-time audio manipulation.

## Installation

`;install https://github.com/jamminstein/prisma`

## Controls

- **E1**: Navigate effect pages (6 total)
- **E2**: Main effect parameter
- **K3**: Smart randomize (generates interesting variations)

## Effects

1. **Granular Morph**: Morph between granular cloud densities
2. **Spectral Freeze**: Freeze and manipulate spectral content
3. **Reactive Grain**: Grain generation driven by input amplitude
4. **Modal Resonator**: Sympathetic resonance and modal filtering
5. **Grain Clouds**: Dense granular cloud generation
6. **Spectral Shift**: Frequency domain manipulation

## Features

- 6 unique spectral and granular effects
- One parameter per effect for intuitive control
- Smart randomization for creative exploration
- Real-time audio processing
- Minimal interface, maximum impact

## Requirements

- norns
- SuperCollider engine (Engine_PRISMA.sc)
- Audio input/output
